Tích hợp dịch vụ vận chuyển Shopee Xpress với WooCommerce. Tự động tính phí giao hàng qua API Shopee.

== Description ==

Shopee Xpress Shipping Integration là một plugin giúp bạn:
- Tự động tính phí giao hàng thông qua API Shopee Xpress.
- Cấu hình trực tiếp API Key, Merchant ID và URL từ WooCommerce admin.
- Hỗ trợ nhiều vùng vận chuyển (Shipping Zones).
- Dễ dàng cập nhật thông qua GitHub nhờ tích hợp Plugin Update Checker.

== Installation ==

1. Tải plugin dưới dạng `.zip` và cài đặt qua WordPress Admin > Plugins > Add New > Upload Plugin.
2. Kích hoạt plugin.
3. Truy cập **WooCommerce > Settings > Shipping**, chọn vùng vận chuyển và thêm phương thức **Shopee Xpress**.
4. Cấu hình `API Key`, `Merchant ID`, và `Shipping API URL`.
